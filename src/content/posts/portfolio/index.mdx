# Propósito
El porfolio es un proyecto muy útil para los desarrolladores. Funciona principalmente como carta de presentación, 
expositor de proyectos y habilidades técnicas y contacto con el entorno profesional. 
Es muy interesante complementarlo con un blog u otra forma de contenido, donde profundizar sobre los proyectos 
y publicar actividades relacionadas con la actividad profesional.

# Diseño
Para el diseño se ha buscado un estilo limpio y moderno, inspirado en las tendencias
actuales. Se ha utilizado una paleta de colores sencilla que permita una visualización
clara del contenido. Se ha implementado un modo claro y oscuro, permitiendo al usuario
elegir según sus preferencias de lectura. Además, se ha adoptado un diseño adaptativo
que asegura una experiencia de usuario óptima en cualquier tamaño de pantalla.

# Tecnologías

El diseño se ha realizado mediante Figma, una aplicación de diseño y prototipado.
Para el desarrollo he optado por el framework web Astro. Ideal para el desarrollo de porfolios, 
por su requisito técnico de uso básico de HTML, CSS y su buen rendimiento en páginas centradas en el contenido.
Para el estilo opté por usar TailwindCSS, un framework de CSS que permite mediante nombres de clases 
preestablecidas estilar el contenido. Permite una coherencia de estilos al usar sus clases y un desarrollo rápido.
